Interactive Orbit Portfolio — Project Specification
Project Overview

Create a cinematic, futuristic single-page portfolio landing experience inspired by atomic/orbital motion visuals.

The center of the screen contains a glowing non-clickable core sphere representing the creator identity. Orbiting around it are 3 animated glowing spheres:

FullStack
Game Development
Multimedia Production

Each orbiting sphere is clickable and redirects to its own portfolio section/page.

The visual style should feel:

Minimal but premium
Sci-fi / holographic
Smooth and atmospheric
Elegant motion design
Soft glow, blur, bloom, and fading transitions
Responsive on desktop and mobile

The goal is to make the homepage itself feel like an interactive animated identity rather than a traditional portfolio.

Main Scene

The page opens with:

Dark deep-space style background
Subtle animated particles/stars
A glowing center sphere
3 orbiting animated spheres
Orbit trails rendered dynamically

The orbiting spheres move continuously around the center in smooth elliptical paths.

The center sphere is NOT clickable.
